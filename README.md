# Roblox-Killer 

### Setup

1. Open requirments.bat
2. Put the roblox account cookie in settings.json
3. If the cookie is from someone else, Change `Bypass` to `true`
4. Install the packages in requirements.txt if you don't have them
5. Run start.bat
